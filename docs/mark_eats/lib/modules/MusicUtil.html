generate_scale (root_num, scale_type[, octaves])
Generate scale from a root note.

Parameters:

  • root_num MIDI note number (0-127) where scale will begin.
  • scale_type String defining scale type (eg, "major", "aeolian" or "neapolitan major"), see class for full list.
  • octaves Number of octaves to return, defaults to 1. (optional)

Returns:

    Array of MIDI note numbers.
snap_note_to_array (note_num, snap_array)
Snap a MIDI note number to the nearest note number in an array.

Parameters:

  • note_num MIDI note number input (0-127).
  • snap_array Array of MIDI note numbers to snap to, must be in low to high order.

Returns:

    Adjusted note number.
snap_notes_to_array (note_nums_array, snap_array)
Snap an array of MIDI note numbers to an array of note numbers.

Parameters:

  • note_nums_array Array of input MIDI note numbers.
  • snap_array Array of MIDI note numbers to snap to, must be in low to high order.

Returns:

    Array of adjusted note numbers.
note_num_to_name (note_num[, include_octave])
Return a MIDI note number's note name.

Parameters:

  • note_num MIDI note number (0-127).
  • include_octave Include octave number in return string if set to true. (optional)

Returns:

    Name string (eg, "C#3").
